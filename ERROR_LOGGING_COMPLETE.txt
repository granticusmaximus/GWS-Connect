## 🚀 Setup (2 steps)

### Step 1: Environment Variables
```env
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
SMTP_SECURE=false
SMTP_USER=your-email@gmail.com
SMTP_PASS=your-app-password
SMTP_FROM=noreply@gws-connect.local
SMTP_REPLY_TO=support@gwsapp.net
```

### Step 2: Set Recipient Email
Edit `server/src/services/errorReporter.js`:
```javascript
const ERROR_RECIPIENT = 'your-email@example.com';  // ← Change this
```

**That's it!** System is ready. ✅

## 💡 Usage Examples

### Route Handler
```javascript
try {
  // Your code
} catch (error) {
  await logError(error, {
    file: 'routes/messages.js',
    function: 'POST /api/messages',
    operation: 'Creating message',
    userId: req.user?.id,
  });
}
```

### Socket Handler
```javascript
socket.on('event', async (data) => {
  try {
    // Socket code
  } catch (error) {
    await logSocketError(error, {
      file: 'index.js',
      socketId: socket.id,
      userId: socket.user.id,
      eventName: 'event',
      operation: 'Processing socket event',
    });
  }
});
```
